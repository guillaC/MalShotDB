## Directory Structure
Bin/release-debug directory is composed of 4 directories:

- Extracted : All ressource files (ICO..)
- Malwares : Executables (.exe only) launched one by one
- Screenshots
- Tools : [Manalyze binary](https://manalyzer.org/)
